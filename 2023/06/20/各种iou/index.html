各种iou

发表于 2023-06-20 阅读次数：

记录一下各种IoU

IoU loss

bounding box重叠部分的面积/并集的面积

缺点：

  • IoU=0的时候，距离近的框和距离远的框损失一致，理论上越近越小才对
  • IoU相等的时候，损失函数一致，但是无法对不同位置的框区分

GIoU loss

GIOU

如图所示，找到最小外接矩形，然后通过计算蓝色的地方的面积来进行计算loss，蓝色地方越大，损失函数越大。 \[giou = IOU-\frac{A_{c}-U}{A_{c}}\] 其中，\[A_{c}-U\]就是两个蓝色框的面积， 如图所示

GIOU2

缺点

当真实框包含黄色框的时候，会退化成IOU，还是没有解决相对位置的问题

DIoU

在GIOU的基础上增加了相对位置的信息，将预测框的中心点与真实框中心点进行距离判断,综合考虑了目标与anchor之间的距离，重叠率以及尺度 \[DIoU = IoU-\frac{\rho^{2}(b,b^{gt})}{c^{2}}\] 其中\(b,b^{gt}\)是预测框和真实框的中心点，\(\rho\)是欧式距离，c是能外接框的对角线距离

优点

  • 在框不重叠的时候，可以提供框的移动方向
  • 可以直接最小化和目标框的距离，收敛快
  • 两个框在同一水平或垂直方向的时候，收敛的快，GIoU则会退化成IOU
  • 可以替换IOU的评估，应用在NMS里面，使得NMS的结果更合理有效

CIoU

yolov5 里边使用的是CIoU loss， CIoU loss 考虑的是回归框宽高比例以及真实框与预测框中间的距离 \[R_{CIoU} = \frac{\rho^{2}(b, b^{gt})}{c^{2}}+\alpha v\] \[L_{CIoU} = 1-IoU+R_{CIoU}\] 其中\(\alpha\)是权重参数，v是长宽比的相似性定义为\(v=\frac{4}{\pi ^{2}}(arctan \frac{w_{gt}}{h_{gt}}-arctan \frac{w}{h})\) 最后，CIoU loss的梯度类似于DIoU loss，但还要考虑v的梯度。在长宽在[0,1]的情况下，\(w^{2}+h^{2}\)的值通常很小，会导致梯度爆炸，因此在\(\frac{1}{w^{2}+h^{2}}\)实现时将替换成1。

EIoU

将原始的宽高比例，改为宽高值回归，将红色框的宽度与真实（蓝色）宽度进行损失，红色长度与真实长度进行回归。

CIOU loss= IOUloss+中心点损失+长宽比例损失

改为

EIOU loss =IOUloss+中心点损失+宽损失+长损失

作者认为或许可以加上Focal loss，并认为简单的任务框回归不需要用过大的权重来学习，而复杂的任务框回归需要大权重来学习，进而将EIOU loss 进行改进提出 Focal EIOU loss 公式见下（这里有个focal l1 loss 推导） \[L_{focal-EIoU} = IoU^{\gamma}L_{EIoU}\] \(\gamma=0.5\)时最优，可以简单解读当任务很简单的时候IOU接近为1退化为EIOU loss，当任务很难时候IOU值很小，然后EIOU的值会变大但是IOU的前面值小于1说明变换小于EIOUloss变换，因此遇到难的任务时loss会增大，增加对难的回归样本的学习。

这里有个问题就是，我当时想，hard label不是iou小的样本么？不是应该要乘一个大系数么？实际上看下图：

SIoU

该作者认为上述框都没有考虑角度问题，但是方向框的角度确实可以影响回归，因此提出SIoU loss，其中SIoU使用4组cost组成见下

SIoU =Distance cost（angle + distance）+Shape cost+IOU loss

siou

Angle cost

\[A = 1-2*sin^{2}(arcsin(x)-\frac{\pi}{4})\] 其中\(x=\frac{c_{h}}{\rho}=sin(\alpha)\)

化简后\(A = sin(2\alpha)\),即损失就是当前角度的2倍取sin值，在0度的时候直接向左回归即可，在alpha=pi/4时取sin（pi/4*2）=1为最大值，alpha<pi/4。 这一惩罚可以引导anchor box移动到目标框的最近的轴上

Distance cost

描述的是中心点之间的距离， \[d = \sum_{t=x,y}(1-e^{-\gamma \rho _{t}})\] 其中\(\rho_x = (\frac{b_{cx}^{gx}-b_{cx}}{c_{w}})\),\(\rho _{y}\)同理，是中心点间的距离，\(\gamma = 2-A\) 其中\(\rho_t\)是距离损失使用2次幂来赋权重。说明距离的影响要大于角度的影响，同时定性来分析，当角度为45°，\(\gamma\)为1，角度损失增大。角度为0时候\(\gamma\)=2，退化为距离损失。他和角度损失是正相关的。

shape cost

\[S = \sum_{t=w,h}(1-e^{\omega_{t}})^{\theta}\] 其中\(\omega_{w}=\frac{|w-w^{gt}|}{max(w, w^{gt})}, \omega _{h}\)同理 就是从长宽来看现在的回归框的形状与标签框是否相似

θ是个可调变量，来表示网络需要对形状给予多少注意力，即占多少权重。实验中设置为4。
