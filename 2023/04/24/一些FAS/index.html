    DATASETS

    我这里用到的常用数据集一共有4个，OULU-NPU, CASIA-FASD, MSU-FASD和IDIAP REPLAY-ATTACK。

    项目地址：OULU-NPU - a mobile face presentation attack database with real-world variations

    • 场景一：在一个开放式的办公室里拍摄的，在那里，电子灯被打开，百叶窗被打开，窗户位于用户的后面。
    • 场景二：是在一间会议室里拍摄的，那里的电子灯是唯一的照明光源。
    • 场景三：是在一个小办公室里拍摄的，在那里电子灯被打开，百叶窗被打开，窗户就在用户面前。

    采集设备

    1. 三星Galaxy S6 edge（手机1），配备500万像素的前置摄像头。
    2. HTC Desire EYE（手机2），拥有1300万像素的前置摄像头。
    3. MEIZU X5（手机3），配备500万像素的前置摄像头。
    4. ASUS Zenfone Selfie（手机4），拥有1300万像素的前置摄像头。
    5. 索尼XPERIA C5 Ultra Dual双摄像头（手机5），1300万像素前置摄像头。
    6. OPPON3（电话6），配备1600万像素的旋转摄像头。

    全高清分辨率录制的，即1920×1080，使用了6个移动设备的前置摄像头和安装在每个设备上的相同的摄像头软件。尽管一些移动设备的名义相机分辨率是相同的，比如索尼XPERIA C5超双版，HTC Desire EYE和ASUS Zenfone自拍（1300万像素），但在结果视频的质量上可以观察到显著的差异（手机厂对拍照的优化算法的不同）

    攻击的来源都是用三星s6 edge的后置摄像头拍摄的（1600万像素 ）

    所以数据集的格式是：采集设备_采集场景 _ID_攻击类型

    CASIA-FASD(2012)

    论文地址：A face anti-spooing database with diverse attacks

    项目地址：A face anti-spooing database with diverse attacks

    casia_sample

    如图所示是一个完整的视频集。左图上四张图片代表低质量视频，左下角为正常质量视频，右图为高质量视频。对于每一种质量，从左到右都是真实的，扭曲的照片攻击，剪切照片攻击和视频攻击。

    采集设备

    使用三个不同的相机来记录不同质量的数据。

    低质量的视频是由一个长期使用的USB相机捕获的，因为长期使用总是会降低成像质量。低质量视频的图像分辨率是640×480。

    正常质量的视频是由一个新购买的USB相机拍摄的，它可以很好地保持原始的成像质量。分辨率是450×640.

    高质量的视频是使用一个高分辨率的索尼NEX-5相机进行拍摄的，其最大分辨率高达1920×1080。我们为每个受试者拍摄一张最大分辨率为1920×1080的图像作为制作照片的材料。然而，对于真实的面部视频和视频攻击，如此高分辨率的视频对于保存和计算都是一个太沉重的负担。因此，我们只裁剪一个1280×720图像补丁，以形成最终的视频。通过这样做，我们减少了节省和计算的负担，同时最大化了我们可以实现的质量。图中的一些例子见图3，其中只显示了面部部分。

    共50名用户，没有人工环境的统一，被试被要求表现出眨眼的行为，而不是保持静止。

    攻击手段

    都是使用索尼NEX-5相机为每个受试者记录一个1920×1080图像和一个1280×720视频作为攻击源。

    1、扭曲的照片攻击(Warped photo attack):

    我们用这种高分辨率的图像来打印照片，这些照片被打印在相纸（铜纸？）上，其质量比普通的A4打印纸要高得多。在一次扭曲的照片攻击中，攻击者故意扭曲一张完整的照片，试图模拟面部运动。完整意味着在照片中没有cut-off区域，这与下面的照片切割攻击相反。我们的扭曲照片攻击与图2中的非常相似。

    2、剪切照片攻击(Cut photo attack.):

    为了表现出眨眼的行为，这里的眼睛区域被切断（cut-off）了。攻击者躲在后面。另一种可能的实现是将一张完整的照片紧紧地放在切割照片的后面，通过在后面移动照片，可以模拟眨眼。在这个数据库中，同时存在这两种实现。

    3、视频攻击（Video attack. ）：

    使用iPad显示高分辨率的真实视频。请注意，受iPad屏幕分辨率的限制，原始的高分辨率视频（1280×720）将不可避免地被该设备缩小。

    测试协议

    数据命名： 精度_攻击类型，精度用L、N、H表示低质量，正常质量、高质量，攻击类型用1、2、3、4表示真实人脸，wpa，cpa，va

    数据库主要考虑了成像质量和各种假脸的可能影响，因此有必要通过验证不同场景下的算法来进行全面的分析。具体来说，我们设计了一个由7个场景组成的测试协议，每个场景只涉及特定的示例。符号L、N、H表示为低质量、质量正常、高质量。

    1、质量测试。我们明确地考虑了这三种成像质量。具体来说，所使用的样本是：a 低质量测试：仅使用{L1、L2、L3、L4}。 b 正常质量测试：仅使用{N1、N2、N3、N4}。 c 高质量检测：仅使用{H1、H2、H3、H4}。

    2、攻击测试。类似地，这三种假脸类型也被明确地考虑过了。所使用的样品有：d 扭曲照片攻击测试：{L1、N1、H1、L2、N2、H2}。e 剪切照片攻击测试：{L1、N1、H1、L3、N3、H3}。 f 视频攻击测试：{L1、N1、H1、L4、N4、H4}。

    3、总体测试。在这种情况下，将所有的数据组合在一起，以给出一个一般的和全面的评估。

    IDIAP REPLAY-ATTACK(2012)

    论文地址：On the effffectiveness of local binary patterns in face anti-spoofifing.

    项目地址：On the effffectiveness of local binary patterns in face anti-spoofifing.

    REPLAY-ATTACK数据库由对50个人的真实访问和攻击尝试的短视频记录组成。总的来说 应该是用了两个采集设备在两个场景下采集人脸，这个就直接拿来当real了，然后是把高清的图片打印出来，有两个场景；把iPhone采集照片、视频的放在iPhone上，平板采集的照片、视频放在平板上呈现给MacBook的摄像头，然后有两个场景以及fixed和hand

    所以一个subject有4个real、4个print、8个iPhone replay、8个iPad replay一共24个video，50个人就有1200个video

    采集设备

    每个人在两种不同的静止条件下录制了多个视频：

    (1)固定（场景的背景均匀，荧光灯的光线照亮场景）；

    (2)变化（场景的背景不均匀，日光照亮场景）。人们被要求坐在苹果13英寸MacBook笔记本电脑上的定制采集系统前，捕捉两个分辨率为320x240像素（QVGA）的视频序列，每秒为25帧，15秒（375帧）。

    在相同的照明和背景设置下，对每个人用1210万像素的佳能PowerShot SX150相机和iPhone3GS（310万像素）拍两张高清照片，然后拿来打印或者重放视频

    攻击手段

    (1)打印（操作员使用凯旋-ADCC2520彩色激光打印机打印的高分辨率的数码照片）；

    (2)手机重放（操作员使用iPhone屏幕显示用iPhone拍摄的照片和视频）；

    (3)高清重放(操作员使用高分辨率的iPad（1024*768像素）显示高分辨率数码照片和视频）。

    每个攻击视频以两种不同的攻击模式捕获约10秒：(1)手持（操作员自己手持攻击媒体或设备）；(2)固定支持攻击（操作员将攻击设备设置在固定支架上，使其在欺骗尝试时不会移动）。手持攻击显示出一种震动行为，有时会欺骗眼睛闪烁的探测器[PWS08]。图1显示了被捕获的欺骗尝试的一些帧。

    EVALUATION

    常用的测试的评价指标有：

    APCER、BPCER、ACER、HTER、EER、AUC

    下面给出一些计算代码

    APCER、BPCER、ACER

    APCER(Attack Presentation Classification Error Rate，攻击呈现分类错误率）

    \(APCER = \frac{1}{N_{PAI}} \sum_{i=1}^{N_{PAI}}(1-Res_{i})\)其中，\(N_{PAI}\)是假体的总共技数量，\(Res_{i}\)为第i次判断的结果，如果是真实人脸就是0，否则是1

    也就是APCER是评价误分类成真实人脸的攻击人脸占攻击人脸的比例

    同理BPCER(Bona Fide Presentation Classification Error Rate,真实呈现错误率)则为：

    \(BPCER = \sum_{i=1} ^{N_{BF}} \frac{Res_{i}}{N_{BF}}\)，其中\(N_{BF}\)就是真实人脸的数目

    而ACER就是他两的一半

    EER和HTER

    常见的spoof traces

    硬件的区别：

    红外摄像头、多光谱摄像头是最常用的一种. 不同波段的红外如近红外、短波红外、热红外都有相关研究工作. 假体人脸的材质与活体人脸的皮肤、眼睛、嘴唇、眉毛等部位的材质不同, 而材质的不同就会造成反射属性有差异. 虽然假体人脸在可见光条件下看起来和活体人脸非常相似, 但是在红外光谱下, 活体人脸的皮肤、眼睛、鼻子等区域看起来和假体人脸都有较大差异.

    光场相机：光场相机能够记录光在空间中的方向与位置等信息[106], 文献[21, 73¡74] 利用光场相机拍摄光场照片用于人脸活体检测. 原始的光场照片是由很多个小的微透镜图像组成的. 随着焦距的不一样, 微透镜图像表示不同的光分布. 这些不同光分布的图像能够用于估计输入图像中深度的存在.
