RepOptimizer

发表于 2023-08-10 阅读次数：

前言

算是填坑，之前提到的YOLOv6里边用到了重参数的技巧，这里补充一下

论文地址：https://arxiv.org/abs/2205.15242

代码地址：https://github.com/DingXiaoH/RepOptimizers

概述

一般的RepVGG、MobileOne系列的重参是不利于量化的（学出的参数分布不集中，重参的实好也会有舍入误差），本文提供了一个方法可以直接在训练的时候直接使用直筒型，但是享受到重参的效果，主要是在梯度上动了手脚

先看看网络图

正常的RepConv是如图b一样，3x3卷积+1x1卷积+shortcut，分别BN后相加再relu，重参数的时候把BN参数融合，把1x1Conv padding到3x3再融合，short cut其实就是一个权重为1的1x1卷积

结构化模型重参化训练效率和资源消耗方面表现都比较弱，而且网络进行变换后对模型量化不友好，这限制了结构化模型重参化技术的应用与推广

方法

RepOptimizer的基本流程如Figure2所示，基本包括了4个步骤：

  • 先验知识，比如多分支网络往往比单分支网络精度更高，如RestNet，RepVGG等；
  • 假想先验知识中的网络结构，如图中的CSLA模块；
  • 通过超参数搜索数据集，获得CSLA中的常量；
  • 模拟CLSA中的梯度重参化梯度。

CSLA

\[BN(y) = \alpha \frac{y-\hat y}{\sigma}+\beta\]

在一个多分支结构中，每一个分支只包含一个线性可训练操作或线性可训练操作后接一个常数用以缩放，假设正常的重参模块CSLA有两个卷积核\(W_{A}\)和\(W_{B}\)，X为输入，输出就为：

\[Y_{CSLA} = \alpha_{A}(X*W_{A})+\alpha_{B}(X*W_{B})\]

其中*为卷积操作，\(\alpha_{A/B}\)可以理解为BN，但是BN其实在训练中是非线性的，不利于后续的推理，所以先设置成了一些可学习的参数，这些常数的作用类似于BN的作用，常数的目标其实就是想搜出\(\frac{\alpha}{\sigma}\)和\(\frac{\alpha\hat y}{\sigma}+\beta\)。

CSLA使用SGD进行梯度更新的时候是：

\[\alpha_{A} W^{i+1}_{A} + \alpha_{B} W^{i+1}_{B} =\alpha_{A} W^{i}_{A} + \alpha_{B} W^{i}_{B}-\gamma(\alpha_{A}\frac{d L}{d_{W^{i}_{A}}}+\alpha_{B}\frac{d L}{d_{W^{i}_{B}}})\]

GR

我们的目标是找到梯度的变换，让我们可以使用直筒的训练结构得到重参数的效果，所以网络结构是GR，使用\(W^{'}\)表示卷积核参数（这里用了‘搞得我一开始以为是梯度..）那么有

\[Y_{GR} = X*W^{'}\]

所以目标是找到优化算法F，使得\(Y_{GR}\)一直等效于\(Y_{CSLA}\)，也就是：

\[W^{'(i+1)} = W^{'(i)}-\gamma F(\frac{d L}{dW^{'(i)}})\]

Grad Mask

实际算梯度的时候会初始化一个grad mask\(M_{c,d,p,q}\in R^{C\times C\times 3\times 3}\)，形式是这样的: \[ \begin{equation} M_{c,d,p,q} = \left\{ \begin{aligned} 1+s_{c}^{2}+t_{c}^{2}, && if c=d,p=2, q=2 \\ s_{c}^{2}+t_{c}^{2}, && if c\neq d,p=2, q=2 \\ s_{c}^{2}, &&otherwise \end{aligned} \right . \end{equation} \]

其中3x3是卷积核大小，p和q是卷积核参数的行列index，CxC是输入输出的通道数，\(s_{c}^{2}\)和\(t_{c}^{2}\)分别是3x3Conv和1x1Conv搜出的参数，按上面说的，对于CSLA重参的时候对于shortcut，和1x1卷积在融合的时候是进行padding的，那其实对于梯度的影响只会发生在卷积核中间，即p=q=2的时候，只有在对应的通道下(c=d的时候)shortcut的梯度才会有影响，通道不对应的时候，在卷积核中间以外就只有本身的参数会有影响了，而此时梯度就是\(s_{c}^{2}\),对于卷积核中间就是\(s_{c}^{2}+t_{c}^{2}\)

这里是因为这些搜出来的参数都是常数，实际上就是对卷积结果做了个线性变换，所以梯度就是常数的值
